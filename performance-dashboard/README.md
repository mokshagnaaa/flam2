# Performance Dashboard (scaffold)

This repository contains a minimal scaffold for a high-performance real-time dashboard using Next.js App Router + TypeScript.

Quick start

```powershell
cd c:\Users\moksh\Downloads\flam2\performance-dashboard
npm install
npm run dev
```

Files of interest
- `app/dashboard` — main page and layout
- `components/charts/LineChart.tsx` — canvas-based line chart placeholder
- `lib/dataGenerator.ts` — initial data generator
- `app/api/data/route.ts` — example API route
